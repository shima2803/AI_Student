# Dashboard | IA na Vida dos Estudantes

## Sobre o projeto
Este dashboard foi desenvolvido em Power BI com base no dataset **How AI is Changing Student Life**, com o objetivo de analisar como ferramentas de inteligência artificial estão impactando a rotina dos estudantes.

A proposta do painel é apresentar, de forma visual e objetiva, padrões de uso da IA no contexto estudantil, destacando perfil acadêmico, finalidade de uso, satisfação e tempo médio de utilização.

## Imagem do Dashboard
![Dashboard IA - Estudantes](./image/IA%20-%20Estudantes.png)

## Principais análises do dashboard
O painel traz uma visão geral sobre o uso da inteligência artificial entre estudantes, incluindo:

- Distribuição dos respondentes por nível de escolaridade
- Ferramentas de IA mais utilizadas
- Motivos de uso das ferramentas
- Nível de satisfação com diferentes IAs
- Média de horas de uso por idade

## Insights observados
A partir da construção do dashboard, alguns pontos chamam atenção:

- **Escola** foi a categoria com maior número de respondentes, seguida por **Ensino Médio** e **Universidade**
- **Gemini** e **Grammarly** aparecem entre as ferramentas com maior volume total de uso no painel
- **ChatGPT** se destacou no uso voltado para **desenvolvimento de códigos**
- O gráfico de satisfação permite comparar como cada ferramenta é percebida entre os níveis **Low**, **Medium** e **High**
- A média de horas por idade mostra variações relevantes entre diferentes faixas etárias

## Objetivo do projeto
Este projeto foi criado com foco em prática de análise de dados e desenvolvimento de dashboards, buscando transformar dados brutos em informações visuais mais claras, úteis e interpretáveis.
